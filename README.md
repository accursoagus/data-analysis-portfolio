# Portafolio de Análisis de Datos y Estadística

Este repositorio compila proyectos académicos enfocados en el Análisis Exploratorio de Datos (EDA), Inferencia Estadística y Visualización, desarrollados en el marco de la Tecnicatura Universitaria en Inteligencia Artificial (FCEIA - UNR).

El objetivo principal es demostrar la capacidad de procesar datasets crudos (reales y públicos), aplicar metodologías estadísticas rigurosas y extraer conclusiones fundamentadas utilizando tanto Python como R.

## Contenido del Repositorio

### 1. Análisis de Violencia de Género (Línea 144)
Estudio exploratorio sobre los llamados a la línea 144 en Argentina, enfocado en la distribución temporal y geográfica de las denuncias.
* **Herramientas:** Python, Pandas, Matplotlib.
* **Técnicas:** Limpieza de datos (Data Cleaning), tratamiento de valores faltantes, agrupación temporal y visualización de distribuciones categóricas.
* **Archivo principal:** `01-analisis-violencia-genero/analisis_144.ipynb`.

### 2. Inferencia Estadística en la EPH (INDEC)
Análisis socioeconómico basado en la Encuesta Permanente de Hogares (EPH) del INDEC. Se investigaron brechas salariales y niveles educativos mediante herramientas de estadística inferencial.
* **Herramientas:** Lenguaje R.
* **Técnicas:**
    * Análisis descriptivo de variables poblacionales.
    * Pruebas de hipótesis (Test de Student, Chi-Cuadrado) para validar diferencias significativas en ingresos según género y nivel educativo.
    * Verificación de supuestos (Test de Normalidad de Shapiro-Wilk).
* **Nota:** El archivo `informe_parcial.pdf` contiene las conclusiones de la primera etapa, mientras que el script `script_analisis.R` incluye la implementación completa de los tests estadísticos avanzados.

### 3. Accidentología Vial en Santa Fe
Análisis de siniestros viales en la provincia de Santa Fe, integrando datos tabulares con información geoespacial.
* **Herramientas:** Python, Pandas, JSON (Geo).
* **Técnicas:** Manipulación de archivos JSON para mapeo de coordenadas, análisis de frecuencias de siniestros y correlación con factores externos.
* **Archivo principal:** `03-accidentes-transito-santafe/analisis_accidentes.ipynb`.
